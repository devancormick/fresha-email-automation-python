# Changelog

## [1.1.0] - 2026-01-15

### Added
- **Retry Logic**: Automatic retry for email sending with exponential backoff
- **Health Checks**: Comprehensive system health monitoring
- **CLI Interface**: Command-line tools for management and operations
- **Metrics Collection**: Email statistics and success rate tracking
- **Database Backup**: Automatic daily backups with retention policy
- **Rate Limiting**: Protection against too many requests
- **Configuration Validation**: Pydantic-based config validation
- **Deployment Scripts**: Automated deployment and backup scripts
- **Enhanced Logging**: Better error tracking and monitoring
- **Scheduler Improvements**: Job event listeners and automatic backups

### Enhanced
- Email service with retry mechanism
- Scheduler with health checks and backups
- Error handling and recovery
- Documentation and deployment guides

### Fixed
- Improved error messages
- Better exception handling
- More robust database operations
